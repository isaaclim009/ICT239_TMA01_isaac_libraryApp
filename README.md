# SG Library App Setup Guide

## 1. Clone the Repository
```bash
git clone <repository-url>
cd ICT239_TMA01_isaac_libraryApp
```
## 2. Set Up Virtual Environment
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 3. Run the App
```bash
flask --debug run
```
